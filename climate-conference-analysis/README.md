# Climate Conference Project

## Project Overview
This SQL project analyzes conference data for a climate research organization that is hosting a large event with hundreds of attendees. The focus of this project is on managing 30 VIP members, including their event participation, hotel reservations, and RSVP status for special conference events.

The goal of this project is to answer business-style questions using SQL, similar to what a data analyst would do in a real work environment. The analysis uses multiple tables and demonstrates joins, aggregations, filtering, calculations, and categorization.

---

## Objective
Answer six business questions using SQL based on the conference dataset.

---

## Dataset Description

This project uses four tables:

### 1. `vips`
Contains information about VIP members, including:
- Member ID
- First and last name
- Assigned speaking event
- Feedback assignment
- Association name
- Association type
- Membership start year

### 2. `reservations`
Contains reservation and RSVP information, including:
- Member ID
- Hotel assignment
- Welcome party RSVP
- Thank you dinner RSVP

### 3. `events`
Contains conference event information, including:
- Event ID
- Event name
- Event type

### 4. `hotels`
Contains hotel information, including:
- Hotel ID
- Hotel name
- Address

---

## Skills Demonstrated
- SQL joins
- Filtering with `WHERE`
- Aggregation with `COUNT`, `SUM`, and `MAX`
- Grouping with `GROUP BY`
- Filtering grouped data with `HAVING`
- Calculations and aliases
- Categorization using `CASE`
- Business-focused data analysis
